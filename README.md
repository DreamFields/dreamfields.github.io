# dreamfields.github.io
My personal blog site

## Scripts

Install dependencies after cloning the repository:

```powershell
yarn setup
```

Start the local Hexo server:

```powershell
yarn server
```

Restart the local server when port 4000 is already in use:

```powershell
yarn restart
```

Start the server for LAN access:

```powershell
yarn server:lan
```

Deploy the generated site to the configured GitHub Pages branch:

```powershell
yarn deploy
```
